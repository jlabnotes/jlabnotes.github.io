Calculate math result of two drop down values

Posted Sep 12, 2020 Updated Dec 25, 2023
By Bogdan Gorka
1 min read

Use case

I have two custom field fields:

  • Impact, type: select List (single choice) with values:
    • 1-Very Low
    • 2-Low
    • 3-Medium
    • 4-High
    • 5-Very High
  • Probability, type: select List (single choice) with values:
    • 1-Very Low
    • 2-Low
    • 3-Medium
    • 4-High
    • 5-Very High

I have the third field - Risk Rating, type: Number Field

I want to send the result of calculations of Impact and Probability to the Risk rating field

Solution

Smart value:

1

 {{#=}}{{issue.probability.value.left(1)}} * {{issue.impact.value.left(1)}}{{/}}

value.left means to cut off the first character from the drop-down value.

Automation config

Use the smart value in the Edit Issue action.

Result
